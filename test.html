Agent Tracker — Quick Instructions

Monthly workflow. Local-only storage. Import-first setup.

Best setup: Import a previous month that includes your full team, then click New Month. That keeps agents + requirements and wipes records clean.

1) First-Time Setup

  1. Import last month’s BPA report (full team).
  2. Click New Month to clear records but keep your team.
  3. If you don’t have a prior month, you can add agents manually.
Tip: You only need to set the side-by-side requirement once per agent.

2) Import BPA Report (Important)

  1. Download the BPA report.
  2. Open it in Excel.
  3. Save As data.xlsx (this step is required).
  4. Click Import BPA Report and select data.xlsx.
BPA exports are often janky. If you skip the Excel re-save, imports may show 0 rows.

3) Add / Edit Records

Click the count pill in any row to manage records for that agent:

  • Coachings — scheduled coaching sessions
  • Sides — side-by-sides
  • Techs — Technical Monitoring (tech monitors)

Inside the modal you can add, edit, or delete entries.

4) Monthly Reset

When a new month starts:

  1. Click New Month.
  2. Confirm the wipe.

This clears all records but keeps agents + requirements.

Storage & Notes

  • Local-only: Data is stored in this browser on this computer.
  • Duplicates: Imports skip duplicates automatically.
  • Search & Sort: Use the toolbar to find agents fast and rank by completion.
